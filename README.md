Project Description:
This project was completed as part of my final assignment for college, it incorporates multiple modules like databases, GUI programming, project management, OOP programming into one big project where all the skills I have learned throughout the year come into practise. The brief for the assignment indicated that the application was to be a record keeping application that stored student personal information and exam results. The current system as stated in the brief was a manual one mainly using written records and excel spreadsheets to store information, the FE college wanted a digital system that made it easier and more efficient to store data. The new system would ultimately be more cost efficient as it reduces the cost of storage for physical records and would staff time when looking for a specific record. The requirements for the project were as follows:
•	A menu-driven application which will allow tutors enter exam results for students as assessments are completed. It should be possible to enter results for a single student or for a class group. 
•	A system which will provide various types of reports, eg a full learner record for a student, results for a particular module for a class group, results for a particular module across all groups, a list of distinction students for a module, a full results report for a class group, etc
•	the system should be user friendly with the minimum amount of input required, eg data entry should be by selection rather than by typing.

Technologies used:
I choose MySQL for my database creation because:
1. Open-Source and Free
•	MySQL is free to use under the GNU license, making it cost-effective for startups and small teams.
2. Reliability and Stability
•	It’s been around for decades and is battle-tested in millions of applications—from blogs to enterprise systems.
3. Cross-Platform Compatibility
•	Works on Windows, macOS, and Linux—ideal for distributed teams and deployment flexibility.

JAVA: I choose JAVA as my programming language because it’s the programming language I am most proficient in, therefore using it will produce the best result for me. Java is also the programming language used on the IDE that I am most confident with i.e. NetBeans.
I choose NetBeans as my IDE because:
1. All-in-One IDE (Integrated Development Environment)
•	NetBeans comes with built-in tools for coding, debugging, compiling, and running applications—no need for extra plugins.
2. Strong Java Support
•	It’s officially supported by Oracle and optimized for Java development (including Java SE, Java EE, and JavaFX).
3. Built-In GUI Builder (Matisse)
•	Makes designing desktop UIs easy with drag-and-drop components—great for visual applications.
4. Cross-Platform
•	Runs on Windows, macOS, and Linux, so teams using different systems can all use the same tool.
